The Automatic Plant Watering Machine System Design Document and the Plant Watering Machine Specification were written as part of an Internet of Things class presented by the University of California-Irvine. The class was hosted on the Edx.org MOOC platform. These documents were written to complete the requirments of the class. The machine itself was never built.
